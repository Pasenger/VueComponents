#### prop参数

 - size ：`'mini'` 为不带掩码 `' ' `空值为带掩码 `可选` 默认空值
 - defValue：这是一个默认值
 - placeholder：掩码输入框的提示文案

##### 组件提供的便利

 1. 判断了输入长度自动跳转下一个ip输入
 2. 判断的回退键`delete`删除到空值在按下自动跳转上一个ip输入
 3. 判断`.`按下时自动跳转下一个ip输入
 4. 组件提供输入`ip校验`与`掩码 0-32`自动转换
